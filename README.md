# 💰 Web3 Donation Platform

A decentralized donation platform that allows users to connect their crypto wallet and send ETH securely.

## 🚀 Features
- Connect wallet using Coinbase Wallet
- Send ETH transactions securely
- Real-time transaction handling
- Simple and user-friendly interface

## 🛠 Tech Stack
- Backend : ethers.js,json,hardhat.config.  
- Web3 Integration: Coinbase Wallet SDK  
- Blockchain: Ethereum  

## ⚙️ How It Works
1. User connects wallet using Coinbase Wallet  
2. Enters donation amount  
3. Confirms transaction  
4. ETH is transferred securely via blockchain
